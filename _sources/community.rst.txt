.. _community:

Community
=========

Updates
^^^^^^^
The `CyberPaleo blog <https://medium.com/cyberpaleo>`_ features updates from the LinkedEarth team and affiliates, as well as guest posts from various partners.

Discussion
^^^^^^^^^^
- Discourse: Announcements and general discussion is done through `this forum <https://discourse.linked.earth/>`_. This is also a great place to ask for help with our tools (e.g. Pyleoclim, GeoChronR, LiPD). This is the best place to start getting involved with LinkedEarth.
- SLACK:  Our Slack workspace is mainly used for hackathons. `Email us <mailto:linkedearth@gmail.com>`_ for an invitation.
- GITHUB: All technical discussion should go through GitHub. Please use project-specific GitHub repositories (e.g. `Pyleoclim repository <https://github.com/LinkedEarth/Pyleoclim_util>`_). All of our packages follow the `Open Development <https://en.wikipedia.org/wiki/Open-source_software_development>`_ model. See each repository for guidelines.

  * `Pyleoclim tutorials <https://www.youtube.com/watch?v=LJaQBFMK2-Q&list=PL93NbaRnKAuF4WpIQf-4y_U4lo-GqcrcW>`_.
  * `LiPD tutorials <https://www.youtube.com/watch?v=6yr-qR0aNCo&list=PL93NbaRnKAuGhl9EcH4fy6H0zcEuqacJO>`_.

PAGES
^^^^^
`Past Global Changes (PAGES) <http://www.pastglobalchanges.org>`_ has supported and advertised LinkedEarth activities since the beginning. Several existing PAGES projects have adopted LinkedEarth tools, including `PAGES 2k <http://www.pastglobalchanges.org/science/wg/acme/intro?id=221>`_, `Iso 2k <http://www.pastglobalchanges.org/science/wg/2k-network/projects/iso2k/intro>`_ and `CRIAS <http://www.pastglobalchanges.org/science/wg/crias/intro>`_.
